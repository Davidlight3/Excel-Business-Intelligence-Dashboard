# Excel Sales Dashboard Analysis

 Project Overview
This project is an interactive Excel dashboard built to analyze sales performance, track KPIs, and uncover business insights from raw sales data using data visualization and reporting techniques.

Business Problem
Businesses often struggle to monitor sales trends, identify top-performing products or regions, and extract meaningful insights from raw data. This dashboard provides a centralized and interactive reporting solution to support decision-making.

Project Objectives
- Analyze sales performance
- Track revenue trends and KPIs
- Identify top-performing products and regions
- Improve business reporting with visual dashboards

Tools & Techniques Used
- Microsoft Excel
- Pivot Tables & Charts
- Slicers
- KPI Metrics
- Data Cleaning
- Data Visualization

Dashboard Features
- Interactive slicers and filters
- KPI summary cards
- Sales trend analysis
- Product and regional performance insights
- Dynamic visual reporting

Key Insights
- Some product categories generated the highest revenue
- Sales performance varied across regions and periods
- Interactive reporting improved data accessibility and analysis

- Data Visualization
- Business Intelligence
